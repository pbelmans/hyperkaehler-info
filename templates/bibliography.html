{# I tried using namespaces to keep track of which items are being cited on a page, but it seems that a global namespace _persists_ through different requests. So one would be perpetually adding more and more keys being cited. Maybe there is a workaround, but for now I'll just explicitly cite the keys, which does cause some redundancies in the way you write a page... #} {% macro cite(keys, optional=False) %} {% if keys is string %}{% set keys = [keys] %}{% endif %} [ {%- if optional -%}{{ optional }}, {% endif %} {% for key in keys %}{{ key }}{% if not loop.last %}, {% endif %}{% endfor %} ] {% endmacro %} {% macro references(keys) %}
References
{% for key in keys %}
{{ key }}
{{ bibliography[key] | safe }} {% endfor %}
{% endmacro %}
