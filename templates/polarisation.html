Polarisation type

Associated to a Lagrangian fibration $X\to\mathbb{P}^n$ one can associate the polarisation type, which describes the polarisation induced by the restriction of an appropriate Kähler class on $X$ to the abelian varieties which appear as generic fiber, see {{ bibliography.cite("MR3556822", "Theorem 1.1") }}.

For {{ macros.OGsix() }}- resp. {{ macros.OGten() }}-type, because the monodromy group is as large as possible, there is only a single type possible. For {{ macros.K3n("n") }}-type there is still only a single type, despite the monodromy not being as large as possible, whilst for {{ macros.Kumn("n") }}-type multiple types can arise.
